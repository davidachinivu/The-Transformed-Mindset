# The Transformed Mindset

A clean-minded, minimal publication platform built for a high school teacher  designed to share mindset, discipline, growth, and life-improvement ideas in a modern, magazine-style format.

This site was created as a personal project with an eye toward clarity, performance, and long-term maintenance. Its structured to be easy to update and extend, while staying fast and accessible to all readers.

---

## What This Project Includes

### Editorial Experience
- **Home page** with featured story and recent posts
- **Article listing** with category filtering and search
- **Individual article pages** with structured content and related suggestions
- **Contact page** with message submission via email

### Architecture & Design
- Built with **Next.js App Router** for modern React routing and pre-rendering
- Styled with **Tailwind CSS** for clean, consistent design
- Uses **TypeScript** for safety and maintainability
- Content is stored as simple data (no CMS required), making it easy to update

### Key Principles
- **Fast loading:** pre-rendered pages, optimized assets
- **Readable typography:** serif headings, clean spacing, high contrast
- **Accessible UI:** semantic markup, proper form labels, keyboard friendly
- **Minimal dependencies:** lightweight build and straightforward structure

---

## Project Layout

This repo contains the full front-end application and minimal server endpoints used to handle contact submissions.

- `app/`  Next.js app routes and pages
- `components/`  UI pieces (cards, navbar, footer, forms)
- `lib/`  content data and utilities
- `data/`  static assets + example content

---

## Why It Was Built

This project was created as a digital showcase for a high school teacher  a place to present thoughtful writing and practical advice in a clean, professional format. Its designed so the teacher can publish new content without worrying about the technical details.

---

## Whats Next

When ready to publish, the site can be deployed anywhere that supports Next.js (Vercel, Netlify, etc.). The content model is intentionally simple, so adding new articles or tweaks can be done without changing the code.
